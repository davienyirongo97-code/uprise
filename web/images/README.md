# Logo Image Instructions

## To add the Uprise Financial Services logo:

1. Save your logo image as: `uprise-logo.png`
2. Place it in this folder: `web/images/`
3. The logo will automatically appear on the login page

## Supported formats:
- PNG (recommended for transparency)
- JPG/JPEG
- SVG

## Recommended size:
- Width: 200-300px
- Height: Auto (maintains aspect ratio)
- The image will be automatically resized to fit

## Current logo path:
`web/images/uprise-logo.png`

If you want to use a different filename, update the src in `web/index.html`:
```html
<img src="images/YOUR-FILENAME.png" alt="Uprise Financial Services" class="logo-image">
```
