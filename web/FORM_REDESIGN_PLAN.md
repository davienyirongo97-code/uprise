# Uprise Finance - Digital Form Redesign Plan

Based on the 7-page paper form, here are the crucial fields to include in the digital version:

## CLIENT REGISTRATION FORM

### 1. Personal Information (Page 1)
- First Name *
- Last Name *
- Title (Mr/Mrs/Miss/Dr)
- Gender * (Male/Female)
- Date of Birth *
- ID Type * (National ID/Passport/Driver's License)
- ID Number *
- Marital Status * (Single/Married/Divorced/Widowed)
- Phone Number *
- Email (optional)

### 2. Location Details (Page 1)
- District *
- T/A (Traditional Authority)
- Home Village *
- Physical Address

### 3. Employment Details (Page 1)
- Name of Employer/Ministry *
- Department
- Employment Number
- Net Monthly Salary (MK) *
- Full Employment Status * (Yes/No)
- Length of Service
- Work Address

### 4. Bank Details (Page 1)
- Bank Name *
- Account Number *
- Account Type * (Savings/Current)
- Branch

### 5. Spouse Information (Page 1 & Page 6)
- Spouse Name
- Spouse Contact

### 6. Witness/Guarantor Details (Page 2 & Page 5)
- Witness Name *
- Witness Contact *
- Relationship to Client *
- Witness Occupation
- Witness Monthly Income

### 7. ID Document Capture
- ID Front Image *
- ID Back Image *

## LOAN APPLICATION FORM

### 1. Loan Details
- Requested Loan Amount (MK) *
- Loan Purpose *
- Repayment Period (auto-calculated as payday)

### 2. Collateral Information (Page 2)
- Collateral Type (dropdown)
- Description
- Serial Number
- Estimated Value (MK)
- (Allow multiple collaterals)

### 3. Terms Acceptance
- Checkbox: I have read and agree to the terms and conditions *
- Digital Signature/Confirmation

## REMOVED FROM DIGITAL VERSION
- Letter of Undertaking (Page 4) - can be uploaded as document
- Guarantor Analysis Form (Page 5) - simplified into witness section
- Spouse Consent Letter (Page 6) - simplified into checkbox
- Loan Structure breakdown (Page 7) - auto-calculated
- Multiple signature sections - replaced with single digital confirmation

## DIGITAL ENHANCEMENTS
- Barcode scanner for quick ID data entry
- Auto-calculation of loan fees and total amount
- Digital signature capture
- Document upload for supporting documents
- Real-time validation
- Progress indicator for multi-step form
